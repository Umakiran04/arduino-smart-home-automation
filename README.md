# 🏠 Smart Home Automation System (Arduino)

## 📌 Overview

This project is an Arduino-based smart home automation system that uses multiple sensors to automatically control home appliances.
It improves energy efficiency, safety, and convenience by monitoring environmental conditions in real-time.

---

## ⚙️ Features

* Automatic control of lights based on ambient light
* Motion detection for security and automation
* Temperature monitoring for environmental awareness
* Real-time sensor-based decision making
* Low-cost and efficient IoT-based solution

---

## 🧰 Components Used

* Arduino UNO
* Temperature Sensor (e.g., LM35 / DHT11)
* LDR (Light Sensor)
* PIR Motion Sensor
* Relay Module / LED
* Jumper Wires
* Breadboard

---

## 🔌 Working Principle

1. Sensors continuously collect environmental data
2. Arduino reads sensor values using ADC (analogRead)
3. Based on conditions:

   * Light ON/OFF using LDR
   * Motion detected → trigger output
   * Temperature monitored
4. Output devices are controlled automatically

---

## 🧠 Technologies Used

* Embedded C (Arduino Programming)
* Sensor Interfacing
* ADC (Analog to Digital Conversion)
* Digital I/O Control
